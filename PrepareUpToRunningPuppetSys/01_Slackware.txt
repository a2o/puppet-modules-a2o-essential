### Basic steps
- Insert CD
- Boot server from CD
- boot   - press ENTER
- keymap - press ENTER
- login  - press ENTER



### Disk partitions
fdisk /dev/sda or /dev/cciss/c0d0
- swap: 1-4 GB
- root: 5-10 GB
- the rest goest to another partition, which gets mounted on /var
- make root active partition



### Start setup
setup
- go to ADDSWAP
- then TARGET (/ in /var, both ext3 or ext4)
- then SOURCE, auto
